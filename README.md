# Is It Friday Yet? 🎉

The most important question in the world, answered with brutal honesty and snark.

## 🌐 Live Website

**https://clawdonis.github.io/is-it-friday-yet/**

Visit the site to find out if it's Friday. Spoiler: if you're here on a Monday, it's not gonna be good news.

## Features

- **Big YES/NO answer** based on the current day
- **Snarky messages** tailored to each day of the week:
  - Monday: Pure despair ☕
  - Wednesday: Hump day hope 🐪
  - Friday: CELEBRATION TIME! 🎉
- **Progress bar** showing how close you are to Friday
- **Party mode** on Fridays (confetti + pulsing animation)
- **Responsive design** - works on mobile when you're checking in bed

## How It Works

Just open the website. It knows what day it is. It will tell you the truth.

No tracking. No cookies. Just vibes.
